# Mario Clone Project

A small Unity clone of Super Mario Bros, aimed to explore game development and proper program design.

## Features
- Player and enemy movement with physics-based jumping
- Ability to collect items that interact with the player/world
- Basic UI and menus

## Controls
- **A&D** or **Left&Right Arrow** - Move
- **Space** - Jump
- **Esc** - Pause

## Getting Started

### Requirements
- Unity 2022.3 LTS
- Windows / macOS

### Running the Project
1. Clone the repository
2. Open Unity Hub
3. Click **Add** -> **Add project from disk**
4. Select the project folder -> Click on the imported project
5. Select Assets -> Scenes -> Main Menu
6. Press Play!
7. Alternatively one can build the entire application: File -> Build Settings -> Build & Run
